# 🚀 מדריך התחלה מהירה - בוט גיוס AIG

## ⚠️ לפני שמתחילים - **חשוב מאוד!**

1. **חשבון בדיקה**: צור חשבון פייסבוק חדש רק לצורך הבוט
   - ❌ **אל תשתמש** בחשבון האישי שלך!
   - ✅ צור חשבון עם שם כמו "ליאב רקרוט" או "גיוס AIG"
   
2. **סיכונים**: יש סיכוי של 50-70% שהחשבון ייחסם תוך 3-6 חודשים

3. **התחל איטי**: 5-10 תגובות ליום לחודש הראשון

## 📝 שלב 1: הכנה (5 דקות)

### 1.1 צור חשבון בדיקה בפייסבוק
- לך ל-facebook.com
- צור חשבון חדש עם אימייל נפרד
- הצטרף לקבוצות היעד (דרושים פתח תקווה, הוד השרון, וכו')

### 1.2 התקנת התלויות
```bash
cd facebook-job-bot
pip install -r requirements.txt
playwright install chromium
```

### 1.3 הגדרת משתני סביבה
```bash
# העתק את קובץ הדוגמה
cp .env.example .env

# ערוך את הקובץ
nano .env

# הוסף את הפרטים:
FB_EMAIL=your_test_account@email.com
FB_PASSWORD=your_test_password
```

## 🔧 שלב 2: עדכון קבוצות (10 דקות)

ערוך את קובץ `config.py` והוסף את הקישורים לקבוצות:

```python
TARGET_GROUPS = [
    {
        "name": "דרושים פתח תקווה",
        "url": "https://www.facebook.com/groups/XXXXXX",  # <- הוסף כאן
        "priority": 1
    },
    {
        "name": "דרושים הוד השרון",
        "url": "https://www.facebook.com/groups/YYYYYY",  # <- הוסף כאן
        "priority": 1
    },
    # הוסף עוד קבוצות...
]
```

**איך למצוא את ה-URL:**
1. היכנס לקבוצה בפייסבוק
2. העתק את הקישור מסרגל הכתובת
3. הדבק ב-config.py

## 📋 שלב 3: עדכון משרות (אופציונלי)

אם אתה רוצה לעדכן את פרטי המשרות, ערוך את `config.py`:

```python
OPEN_POSITIONS = {
    "סוכן ביטוח": {
        "title": "סוכן/ת ביטוח",
        "description": "תיאור המשרה...",
        "requirements": [
            "דרישה 1",
            "דרישה 2",
        ],
        "locations": ["פתח תקווה", "הוד השרון"],
        "keywords": ["מכירות", "שירות", "ביטוח"]
    },
    # עוד משרות...
}
```

## 🧪 שלב 4: בדיקה (5 דקות)

### בדוק שהכל עובד:
```bash
# בדוק מסד נתונים
python database.py

# בדוק מנוע התאמה
python candidate_matcher.py

# בדוק מחולל תגובות
python response_generator.py
```

### הרצת בדיקה אחת:
```bash
python main.py --run-once
```

זה ייפתח דפדפן, יתחבר לפייסבוק, יסרוק קבוצה אחת, ויציג מה הוא מצא.
**אל תדאג - הוא לא ישלח תגובות בפעם הראשונה. צפה מה קורה.**

## 🚀 שלב 5: הפעלה (כשמוכן!)

זה יפעיל את הבוט שירוץ אוטומטית כל 2 שעות בין 9:00-20:00, א'-ה'.

### הצגת סטטיסטיקות:

## 📊 מה הבוט עושה?

1. ⏰ **כל 2 שעות** (9:00-20:00, א'-ה'):
   - נכנס לפייסבוק
   - סורק את הקבוצות שהגדרת
   - מחפש מועמדים (מי שכתב "מחפש עבודה")
   - מתאים אותם למשרות ב-AIG
   - שולח תגובה מותאמת

2. 💾 **שומר הכל במסד נתונים**:
   - אילו פוסטים נסרקו
   - למי הגבנו
   - סטטיסטיקות

3. 🛡️ **טכניקות הסוואה**:
   - עיכובים אקראיים (45-120 שניות)
   - תנועות עכבר אנושיות
   - הקלדה אנושית
   - וריאציות בתגובות

## ⚙️ התאמה אישית

### שינוי כמות תגובות ליום:
ערוך `config.py`:
```python
AUTOMATION_SETTINGS = {
    "max_responses_per_day": 10,  # <- שנה כאן (התחל עם 5-10)
}
```

### שינוי תבניות תגובות:
ערוך `config.py` - `RESPONSE_TEMPLATES`

### שינוי מילות מפתח:
