# מדריך הגדרה - בוט גיוס AIG 🤖

## שלב 1: יצירת חשבון פייסבוק לבוט ⚠️

**חשוב מאוד:** אל תשתמש בחשבון האישי שלך!

1. צור חשבון פייסבוק חדש עם:
   - שם משתמש כמו "AIG Recruitment" או משהו דומה
   - אימייל ייעודי (יכול להיות gmail חדש)
   - סיסמה חזקה

2. **השלם את הפרופיל:**
   - הוסף תמונת פרופיל מקצועית
   - מלא פרטים בסיסיים
   - הוסף מספר חברים (5-10 לפחות)
   - פרסם 2-3 פוסטים רגילים

3. **חכה 2-3 ימים** לפני שמתחיל להשתמש בבוט
   - פייסבוק חושדים בחשבונות חדשים
   - השתמש קצת בחשבון באופן ידני

## שלב 2: הצטרפות לקבוצות 👥

הצטרף לקבוצות הרלוונטיות:

1. חפש קבוצות "דרושים" באזור:
   - דרושים פתח תקווה
   - דרושים הוד השרון
   - דרושים כפר סבא
   - משרות חמות באזור השרון
   - דרושים במרכז

2. **העתק את הקישור של כל קבוצה:**
   - היכנס לקבוצה
   - העתק את ה-URL מהדפדפן
   - דוגמה: `https://www.facebook.com/groups/1234567890`

3. **שמור את הקישורים** - תצטרך אותם בשלב הבא!

## שלב 3: הגדרת משתני סביבה 🔐

1. **צור קובץ `.env`** בתיקיית הפרויקט:
   ```bash
   cp .env.example .env
   ```

2. **ערוך את הקובץ `.env`** ומלא את הפרטים:
   ```
   FB_EMAIL=your_bot_account@example.com
   FB_PASSWORD=your_password_here
   ```

3. **בדוק שהקובץ נוצר:**
   ```bash
   ls -la .env
   ```

## שלב 5: התאמת הגדרות הבוט ⚙️

ב-`config.py`, בדוק והתאם את ההגדרות הבאות:

### מגבלות יומיות (שורה 189-210):
```python
AUTOMATION_SETTINGS = {
    "max_responses_per_day": 10,      # 👈 כמה תגובות מקסימום ליום
    "max_responses_per_hour": 3,      # 👈 כמה תגובות מקסימום לשעה
    "posts_to_scan_per_group": 20,    # 👈 כמה פוסטים לסרוק בכל קבוצה
    "active_hours_start": 9,          # 👈 שעת התחלה
    "active_hours_end": 20,           # 👈 שעת סיום
}
```

**המלצות:**
- התחל עם מספרים נמוכים (5-10 תגובות ליום)
- הגדל בהדרגה לאחר שבוע-שבועיים
- שמור על מראה "אנושי"

### משרות פתוחות (שורה 89-145):
בדוק שכל המשרות מעודכנות:
- סוכן ביטוח
- שירות לקוחות
- חידושים
- תביעות
- שימור

**ערוך לפי הצורך:**
- שנה כותרות
- עדכן דרישות
- שנה מיקומים

### תבניות תגובות (שורה 150-184):
בדוק את 5 התבניות והתאם אותן לסגנון שלך.

**טיפים:**
- השתמש בעברית טבעית
- הימנע מטקסט שיווקי מדי
- הוסף אימוג'י בזהירות
- היה אישי וידידותי

- ✅ הבוט מזהה מועמדים

אם הכל עובד - מצוין! ✨
