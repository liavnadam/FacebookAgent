# 🤖 בוט גיוס עובדים ל-AIG - Facebook Automation

בוט אוטומטי לזיהוי מועמדים פוטנציאליים בקבוצות פייסבוק ותגובה אוטומטית עם הצעות משרה מותאמות.

## ⚠️ אזהרות חשובות

1. **חשבון בדיקה**: השתמש **רק בחשבון בדיקה נפרד**! לעולם אל תשתמש בחשבון האישי שלך.
2. **סיכון חסימה**: יש סיכוי של 50-70% שהחשבון ייחסם תוך 3-6 חודשים.
3. **התחל איטי**: 5-10 תגובות ליום לחודש הראשון, העלה בהדרגה.
4. **פרוקסי**: מומלץ מאוד להשתמש בפרוקסי Residential איכותי.
5. **תנאי שימוש**: אוטומציה זו מפרה את תנאי השימוש של פייסבוק.

## 🚀 התקנה

### 1. התקנת Python ותלויות

```bash
# ודא ש-Python 3.8+ מותקן
python --version

# התקנת תלויות
cd facebook-job-bot
pip install -r requirements.txt

# התקנת דפדפנים של Playwright
playwright install chromium
```

### 2. הגדרת משתני סביבה

```bash
# העתק את קובץ הדוגמה
cp .env.example .env

# ערוך את הקובץ והזן את פרטי החשבון
nano .env
```

### 3. בדיקה ראשונית

```bash
# בדיקת מסד הנתונים
python database.py

# בדיקת מנוע ההתאמה
python candidate_matcher.py

# בדיקת מחולל התגובות
python response_generator.py
```

## 🎯 תכונות

### ✅ מה הבוט עושה

1. **סריקה חכמה**: סורק קבוצות פייסבוק מוגדרות לפי לוח זמנים
2. **זיהוי מועמדים**: מזהה אוטומטית פוסטים של מחפשי עבודה
3. **התאמה**: מתאים כל מועמד למשרה המתאימה ביותר ב-AIG
4. **תגובה**: שולח תגובה מותאמת אישית עם וריאציות
5. **מעקב**: שומר הכל במסד נתונים למעקב וסטטיסטיקות
6. **הסוואה**: שימוש בטכניקות להימנע מזיהוי

### 🎨 משרות פתוחות

הבוט מתאים מועמדים ל-5 משרות עיקריות:
- 💼 סוכן/ת ביטוח
- 📞 נציג/ת שירות לקוחות
- 🔄 נציג/ת חידושי ביטוח
- 📋 מטפל/ת בתביעות
- 🤝 נציג/ת שימור לקוחות

## ⚙️ התאמה אישית

### עריכת קבוצות יעד

ערוך את `config.py`:
```python
TARGET_GROUPS = [
    {
        "name": "דרושים פתח תקווה",
        "url": "https://www.facebook.com/groups/...",
        "priority": 1
    },
    # הוסף קבוצות נוספות...
]
```

### עריכת מילות מפתח

```python
CANDIDATE_KEYWORDS = {
    "positive": [
        "מחפש עבודה",
        # הוסף עוד...
    ],
    "negative": [
        "דרושים",
        # הוסף עוד...
    ]
}
```

### עריכת תבניות תגובות

```python
RESPONSE_TEMPLATES = [
    """שלום {name}! 👋
ראיתי שאת/ה מחפש/ת עבודה...
""",
    # הוסף תבניות נוספות...
]
```

### עריכת הגדרות אוטומציה

```python
AUTOMATION_SETTINGS = {
    "max_responses_per_day": 10,  # שנה לפי הצורך
    "delay_between_responses_min": 45,  # שניות
    # ...
}
```

## 🛡️ טכניקות הסוואה

הבוט משתמש במספר טכניקות למניעת זיהוי:

## 📊 מעקב וניטור

## 🔧 פתרון בעיות

### הבוט לא מתחבר לפייסבוק
- ודא שהאימייל והסיסמה נכונים ב-.env
- נסה להתחבר ידנית לחשבון מהדפדפן
- פייסבוק עשוי לדרוש אימות דו-שלבי

### הבוט מזוהה ונחסם
- העלה את העיכובים ב-config.py
- הפחת את מספר התגובות ליום
- שקול שימוש בפרוקסי Residential

### לא מוצאים מועמדים
- בדוק שהקבוצות ב-TARGET_GROUPS נכונות
- התאם את מילות המפתח ב-CANDIDATE_KEYWORDS
- הרץ `python candidate_matcher.py` לבדיקה

## 🔐 אבטחה

1. **אל תשתף את קובץ .env** - הוא מכיל את הסיסמה שלך
2. **הוסף .env ל-.gitignore** אם משתמש ב-git
3. **השתמש בחשבון בדיקה** - לא בחשבון האישי
4. **גיבויים**: גבה את data/job_bot.db באופן קבוע

## 📞 תמיכה ועדכונים

אם יש בעיות או שאלות:
1. בדוק את הלוגים ב-data/logs/
2. הרץ את הבדיקות: `python database.py`, `python candidate_matcher.py`
3. ודא שכל התלויות מותקנות: `pip install -r requirements.txt`
